# POA - Overcooked
Development of an agent who cooks steaks and fries.

## Initialisation
Cmake (to run after modifying a `CMakeLists.txt` file):
```bash
cmake -S . -B build
```

## Compilation
```bash
cmake --build build
```

## Execution
```bash
./build/src/main
```

## Testing
```bash
cd build/
ctest
```
